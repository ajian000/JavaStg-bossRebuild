# item 目录

## 功能说明

item 目录用于存放游戏中的道具相关实现，包括道具的定义、行为和效果。

## 功能详解

1. **道具定义**:
   - 道具的基本属性
   - 道具的类型和分类

2. **道具行为**:
   - 道具的生成和掉落
   - 道具的移动和碰撞
   - 道具的收集效果

3. **道具效果**:
   - 得分增加
   - 能量恢复
   - 能力提升
   - 特殊效果

## 代码结构

```
item/
└── README.md       # 道具目录说明
```

## 开发建议

- 继承自 `stg.entity.item.Item` 类创建自定义道具
- 实现 `IItem` 接口确保道具功能完整
- 考虑使用对象池管理道具实例，提高性能
