# laser 目录

## 功能说明

laser 目录用于存放游戏中的激光相关实现，包括激光的定义、行为和效果。

## 功能详解

1. **激光定义**:
   - 激光的基本属性
   - 激光的类型和分类

2. **激光行为**:
   - 激光的生成和发射
   - 激光的移动和碰撞
   - 激光的生命周期

3. **激光效果**:
   - 伤害计算
   - 视觉效果
   - 特殊效果

## 代码结构

```
laser/
└── README.md       # 激光目录说明
```

## 开发建议

- 继承自 `stg.entity.laser.Laser` 类创建自定义激光
- 实现激光的视觉效果和伤害逻辑
- 考虑激光的性能优化，特别是对于大型激光效果
