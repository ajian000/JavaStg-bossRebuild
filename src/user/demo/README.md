# demo 目录

## 功能说明

demo 目录包含游戏引擎的演示文件，用于展示各种功能和特性。

## 主要文件

- **ChineseFontDemo.java**: 中文字体演示，展示如何使用中文字体。
- **TriangleMusicDemo.java**: 三角形音乐演示，展示音频播放和图形绘制。

## 功能详解

1. **中文字体演示**:
   - 展示中文字体的加载和使用
   - 测试中文字符的渲染效果

2. **三角形音乐演示**:
   - 展示基本图形绘制（三角形）
   - 演示音频文件的加载和播放

## 代码结构

```
demo/
├── ChineseFontDemo.java      # 中文字体演示
└── TriangleMusicDemo.java    # 三角形音乐演示
```
