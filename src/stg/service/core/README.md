# core 目录

## 功能说明

core 目录包含服务层的核心组件，负责服务的管理和生命周期。

## 主要文件

- **ServiceManager.java**: 服务管理器，负责服务的注册、获取和生命周期管理。

## 功能详解

1. **单例模式**:
   - 使用单例模式确保服务管理器的唯一性
   - 提供全局访问点

2. **服务注册**:
   - 注册服务实例
   - 注册服务提供者

3. **服务获取**:
   - 根据服务类型获取服务实例

4. **服务生命周期**:
   - 初始化所有服务
   - 关闭所有服务

5. **服务管理**:
   - 注销服务
   - 清理服务容器

## 代码结构

```
core/
└── ServiceManager.java    # 服务管理器
```
