# audio 目录

## 功能说明

audio 目录包含游戏的音频服务，负责音频资源的加载、播放和管理。

## 主要文件

- **IAudioManager.java**: 音频管理器接口，定义了音频管理的基本方法。

## 功能详解

1. **音频初始化**:
   - 初始化音频系统
   - 检查音频系统是否已初始化

2. **音频加载**:
   - 加载音乐文件
   - 加载音效文件

3. **音频播放**:
   - 播放音乐（支持循环）
   - 播放音效

4. **音频控制**:
   - 停止音乐
   - 停止所有音效
   - 暂停音乐
   - 恢复音乐

5. **音量管理**:
   - 设置音乐音量
   - 设置音效音量

6. **资源管理**:
   - 卸载音乐
   - 卸载音效
   - 卸载所有音频资源
   - 清理音频系统
