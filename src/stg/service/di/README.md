# di 目录

## 功能说明

di 目录包含依赖注入系统，负责对象的依赖管理和注入。

## 主要文件

- **DependencyContainer.java**: 依赖容器，负责管理服务实例和提供者。

## 功能详解

1. **单例模式**:
   - 使用单例模式确保依赖容器的唯一性
   - 提供全局访问点

2. **服务注册**:
   - 注册服务实例
   - 注册服务提供者

3. **服务获取**:
   - 根据类型获取服务实例
   - 支持延迟初始化（通过服务提供者）

4. **服务管理**:
   - 注销服务
   - 清理所有服务

5. **服务提供者**:
   - 定义服务提供者接口
   - 支持服务的延迟创建

## 代码结构
