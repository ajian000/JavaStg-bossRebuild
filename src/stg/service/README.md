# service 目录

## 功能说明

service 目录包含游戏的服务层，负责提供各种服务功能，包括音频管理、依赖注入等。

## 子目录结构

- **audio/**: 音频服务，负责音频的加载和播放
- **core/**: 核心服务，负责服务的管理和生命周期
- **di/**: 依赖注入服务，负责对象的依赖管理

## 功能详解

1. **服务管理**:
   - 统一的服务注册和管理机制
   - 服务的生命周期管理
   - 服务间的依赖关系处理

2. **音频服务**:
   - 音频资源的加载和管理
   - 音频的播放控制
   - 音频效果的处理

3. **依赖注入**:
   - 对象的依赖管理
   - 依赖的自动注入
   - 单例和多例的管理

## 代码结构

```
service/
├── audio/
│   └── IAudioManager.java     # 音频管理器接口
├── core/
│   └── ServiceManager.java    # 服务管理器
└── di/
    └── DependencyContainer.java # 依赖容器
```
