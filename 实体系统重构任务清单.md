   - [x] 封装旋转方法：
     - [x] rotateTo(angle: Float, time: Float)：旋转到指定角度,耗时time秒,x轴正方向为0,顺时针为正
     - [x] rotate(angle: Float, time: Float)：旋转实体,耗时time秒,x轴正方向为0,顺时针为正
   - [x] 封装加速方法：
     - [x] accelerate(acceleration: Float, time: Float)：以加速度acceleration加速,耗时time秒,acceleration为负数时减速,速度小于0时反向
